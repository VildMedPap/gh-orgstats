# gh-orgstats

## Description
Organisation specific extension for `gh cli` to retrieve different statistics

## Install
```sh
gh extension install VildMedPap/gh-orgstats
```

## Usage
```sh
gh orgstats --org cli --stats repos
# gh orgstats [flags]
```

## Help
```sh
gh orgstats --help
```
